## 使用


> 在启动此组件项目前，假设您已经安装了 scss 依赖，hx 版本 版本为3.0以上。启动此项目须先准备appId。



直接在 hBuilder 中打开项目，在 manifest.json 微信小程序配置 中填入小程序 appId，之后运行到微信开发者工具即可。
